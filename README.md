# Pro-log
## サービスURL
https://pro-log.jp

## サービス概要
プロローグは、味わいと実用性を6軸で数値化し、レーダーチャートで比較できるプロテイン専門レビューサイトです。
既存レビューでは分かりづらい「甘さ・濃さ・溶けやすさ」などの特徴を具体的に可視化できます。ユーザーが自分の好みに合うプロテインを自信を持って選べます。

---

## このサービスへの思い・作りたい理由
私はここ10年間、ほぼ毎朝やトレーニング後にプロテインを摂取していますが、プロテインを買ったあとに以下のようなことを感じたことがあります。

- 「人気だから買ってみたけど思った味と違う」
- 「全然溶けない、ダマになる」

既存のレビューは5段階の星評価や感想ベースでの評価しかなく、**自分の好みに合うかどうかを事前に判断できない**ため、私のような経験をした人も少なくないと思います。

プロテインは1〜3kgで販売されていることが多く、毎日一杯（20g程度）飲む場合でも2ヶ月〜5ヶ月ほど付き合うことになります。
味が合わなかったばかりに余らせて捨ててしまう友人もいました。中には「もうプロテインは買わない」と摂取をやめてしまった人もいます。

せっかく健康やトレーニングのために始めたのに、**商品選びの失敗で挫折してしまうのはもったいない**と感じ、このサービスを作りたいと思いました。

## ユーザー層について
本サービスを利用するユーザーとしては、以下のような方々を想定しています。

### メインターゲット
- **トレーニング／ダイエット／健康目的でプロテインを試してみたい人**

プロテインは市販の飲料ほど美味しいとは限らないため、継続には“自分に合う味”を見つけることが重要です。
特に飲み始めの人が挫折しないよう、**自分に合う一杯を見つけるための情報提供**が本サービスの価値になると考えています。

### サブターゲット
- **毎回違うプロテインを試している人**
- **今まで飲んできたプロテインに飽きてしまい、新しい商品を探している人**

これまで飲んだプロテインと比較しながら商品選定ができるサービスとして活用してもらいたいと考えています。

## サービスの利用イメージ

### 利用の流れ

**1. 商品検索**
トップ画面の検索フォームで「甘め」「濃いめ」などのキーワードを入力して検索します。
検索結果に条件に合ったプロテインが表示されます。

**2. 商品選定**
詳細ページで6つの評価軸（甘さ、濃さ、後味、香り、溶けやすさ、泡立ち）とレビューを確認します。
「自分に合いそう！」と思った商品はブックマークし、比較しながら購入を検討します。

**3. 購入後のレビュー投稿**
実際に飲んだ結果を6軸評価・割り方・コメントとともに投稿します。
他のユーザーから「いいね」が付くことで、「自分のレビューが役に立った」と実感できます。

**4. SNSでのシェア**
レビューをXにシェアして、トレーニング仲間やダイエット仲間に共有できます。

**5. 継続使用**
利用を続けることで自分の好みがより明確になります。
過去レビューを振り返り、リピート商品の検討にも役立ちます。

## ユーザーの獲得について

### MVP段階
- トレーニング初心者やプロテイン未経験者など、知人ベースでユーザーを募り、利用を依頼します。

### その後のアプローチ
1. **SNSでの口コミ拡散**
  レビュー投稿時にXへの共有機能を設け、自然な形で認知を広げます。
2. **SEO対策**
  「プロテイン おすすめ」などの検索キーワードでヒットするよう最適化し、検索流入を増やします。

## サービスの差別化ポイント・推しポイント

### 既存サービスとの比較表

| 既存サービス | 特徴 | 本サービスの差別化ポイント |
|--------------|------|-----------------------------|
| 比較サイト／ランキングサイト／ECレビュー | 多くが汎用的な星評価で、味覚の違いや個人差が判断しづらいです。 | 味の指標（甘さ、濃さ、後味、香り、溶けやすさ、泡立ち）を数値化し、パーソナライズされた比較ができます。 |
| 各社のお試しセット | 実際に飲んで試せますが、コストが高く、ブランド間比較はできません。 | コストを抑えつつ、ブランドを跨いだ比較ができます。 |

---

### このサービスならではの推しポイント

#### **6つの評価軸で「自分好み」を見つけやすい**

既存サービスでは「美味しい：★★★★☆」のような抽象的な評価が多いですが、本サービスでは数値化された評価により、具体的な違いが分かりやすくなります。

**例：**
- 「甘さ：2、濃さ：3、後味：4」 → 甘さ控えめで後味スッキリ
- 「甘さ：5、濃さ：5、後味：2」 → 濃厚で甘めだが後味が残る

→ このように、自分の好みに合わせて検索・比較できます。
